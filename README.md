# Qoyod to Zoho Books Converter

هذا البرنامج هو أداة مصممة للمحاسبين لتحويل ملفات التصدير من نظام "قيود" المحاسبي إلى ملفات استيراد جاهزة للاستخدام في نظام "Zoho Books".

## المتطلبات الأساسية

لتشغيل هذا البرنامج، يجب أن يكون لديك **.NET 8 SDK** مثبتًا على جهازك الذي يعمل بنظام ويندوز 11.

1.  **تحميل .NET 8 SDK:**
    *   اذهب إلى [صفحة تحميل .NET 8 الرسمية](https://dotnet.microsoft.com/download/dotnet/8.0).
    *   ابحث عن قسم **.NET SDK** وقم بتحميل نسخة **x64** لويندوز.
2.  **التثبيت:**
    *   قم بتشغيل الملف الذي تم تحميله واتبع تعليمات التثبيت البسيطة.

للتحقق من أن التثبيت تم بنجاح، افتح شاشة الأوامر (Command Prompt) واكتب الأمر التالي:

```bash
dotnet --version
```

يجب أن يظهر لك رقم الإصدار `8.0.x` أو أعلى.

## خطوات الإعداد والتشغيل

اتبع هذه الخطوات بدقة لتشغيل البرنامج.

### الخطوة 1: تجهيز الملفات

1.  **إنشاء مجلد رئيسي:** قم بإنشاء مجلد جديد على جهازك في مكان يسهل الوصول إليه، على سبيل المثال: `C:\ZohoConverter`.
2.  **نسخ ملفات "قيود":** قم بوضع جميع ملفات الإكسل (`.xlsx`) التي صدرتها من نظام "قيود" داخل هذا المجلد (`C:\ZohoConverter`).

### الخطوة 2: تعديل مسارات الملفات في البرنامج

1.  **افتح ملف `Program.cs`:**
    *   انتقل إلى مجلد المشروع `QoyodZohoConverter.Runner`.
    *   افتح ملف `Program.cs` باستخدام أي محرر نصوص (مثل Notepad).
2.  **تعديل المسار الأساسي:**
    *   في أعلى الملف، ستجد سطراً يشبه هذا:
        ```csharp
        string basePath = @"C:\Path\To\Your\Files"; // <--- REPLACE THIS BASE PATH
        ```
    *   **استبدل** المسار `C:\Path\To\Your\Files` بالمسار الكامل للمجلد الذي أنشأته في الخطوة السابقة. يجب أن يبدو السطر هكذا (كمثال):
        ```csharp
        string basePath = @"C:\ZohoConverter"; // <--- تم التعديل
        ```
    *   **ملاحظة هامة:** تأكد من بقاء علامة `@` في بداية المسار، ومن أن المسار مكتوب داخل علامتي الاقتباس `""`.
3.  **تعديل أسماء ملفات قيود اليومية (إذا لزم الأمر):**
    *   أسفل المسار الأساسي، ستجد قائمة بأسماء ملفات قيود اليومية:
        ```csharp
        IEnumerable<string> journalFilePaths = new string[]
        {
            Path.Combine(basePath, "Journals1.xlsx"),
            Path.Combine(basePath, "Journals2.xlsx")
        };
        ```
    *   قم بتعديل أسماء الملفات (`Journals1.xlsx`, `Journals2.xlsx`) لتطابق الأسماء التي لديك. يمكنك إضافة أو حذف ملفات من هذه القائمة حسب الحاجة.

4.  **حفظ الملف:** احفظ التغييرات التي أجريتها على ملف `Program.cs`.

### الخطوة 3: تشغيل البرنامج

1.  **افتح شاشة الأوامر (Command Prompt):**
    *   اضغط على قائمة "ابدأ" (Start) واكتب `cmd` ثم اضغط Enter.
2.  **الانتقال إلى مجلد المشروع:**
    *   اكتب الأمر `cd` متبوعًا بمسار المجلد الرئيسي للمشروع. على سبيل المثال، إذا كان المشروع موجودًا على سطح المكتب، فسيكون الأمر كالتالي:
        ```bash
        cd C:\Users\YourUsername\Desktop\QoyodZohoConverter
        ```
3.  **تشغيل البرنامج:**
    *   أنت الآن داخل المجلد الرئيسي للمشروع. اكتب الأمر التالي واضغط Enter:
        ```bash
        dotnet run --project QoyodZohoConverter.Runner
        ```

### ماذا سيحدث بعد التشغيل؟

ستنتظر لبضع ثوانٍ، ثم ستعرض شاشة الأوامر تقريرًا مفصلاً عن البيانات (Dry Run). هذا التقرير يوضح لك ملخصًا للملفات وأي أخطاء أو تحذيرات تم العثور عليها قبل عملية التحويل الفعلية.

إذا ظهرت رسالة خطأ تفيد بأن ملفًا ما غير موجود، يرجى التأكد من صحة المسارات وأسماء الملفات في ملف `Program.cs`.
